# Crictension

Crictension is a web application that displays live cricket match information, including match names, statuses, and scores. Users can view ongoing and upcoming matches and add their favorite matches to a favorites list.

## Features

- Display live cricket match information fetched from an API.
- Show ongoing and upcoming matches with detailed information.
- Add and remove matches from the favorites list.
- Receive notifications for favorite matches that are starting soon.

## Technologies Used

- HTML
- CSS
- JavaScript
